## Overview

This project is a full stack e commerce application where anyone can register as seller and sell their product online.There is also dashboard for seller and admin to manage orders, products, view analytics etc.

customers can browse products buy category, filter by price and make payment online.

Admin can view, create, track, and delete all users, orders and products.

The project is build with next js 16.07. Shadcn is used for interactive UI, stipe for payment system, zod with react hook-form for validation, JWT for authentication.

---

## Features

- Authentication (login/logout) functionality
- Interactive dashboard for both admin and sellers
- Dark mode toggle
- product listing with search and filters
- Client-side form validation in TypeScript
- Responsive design powered by shadcn/ui
- payment system integrated with stripe

## Installation

1. Clone the repo
   ```bash
   git clone https://github.com/shaonexplorer/e-commerce-app-front-end.git
   cd your project folder
   ```
2. **Install dependencies:**

   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   # or
   yarn run dev
   ```
